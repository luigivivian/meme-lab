# Phase 4 Plan 01: JWT Route Protection

Add `Depends(get_current_user)` to all unprotected route handlers across 8 route modules.

## Exempt routes (no auth required)
- /health, /status — health checks
- /auth/* — auth endpoints (already handled)
- /drive/health — system health
- /scheduler/health — scheduler status

## Approach
For each of the 8 route files:
1. Add `get_current_user` to the deps import
2. Add `current_user=Depends(get_current_user)` parameter to each route handler
3. Leave health/status endpoints unprotected

## Verification
- `grep -rL "get_current_user" src/api/routes/ | grep -v __` should return only `__init__.py`
- Unauthenticated request to any protected route returns 401
