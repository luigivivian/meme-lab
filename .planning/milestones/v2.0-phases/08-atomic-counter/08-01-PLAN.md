    - Get `today_utc = self._get_pt_today_start_utc()`
    - Dialect-aware upsert per D-08:
      - SQLite: `from sqlalchemy.dialects.sqlite import insert as sqlite_insert`; `stmt = sqlite_insert(ApiUsage).values(...)` then `.on_conflict_do_update(index_elements=["user_id", "service", "tier", "date"], set_={"usage_count": ApiUsage.usage_count + 1})`
      - MySQL: `from sqlalchemy.dialects.mysql import insert as mysql_insert`; `stmt = mysql_insert(ApiUsage).values(...)` then `.on_duplicate_key_update(usage_count=ApiUsage.usage_count + 1)`
    - Values: `user_id=user_id, service=service, tier=tier, date=today_utc, usage_count=1, status=status`
    - After execute, flush, then SELECT to read back current count
    - Return int (current usage_count)

    **`_get_current_count(user_id, service, tier, today_utc)` async method:**
    - SELECT usage_count WHERE user_id + service + tier + date match, status="success"
    - Return 0 if no row exists

    **`check_limit(user_id, service, tier)` async method per D-09:**
    - Get today_utc, current count, limit from get_daily_limit
    - If limit > 0 and current >= limit:
      - Insert a rejected row: plain INSERT (not upsert) with `status="rejected"`, `usage_count=1`, `date=datetime.utcnow()` (full timestamp, not bucketed, to avoid unique constraint collision per research Open Question 1)
      - Return `(False, {"used": current, "limit": limit, "remaining": 0, "resets_at": self._next_pt_midnight_iso()})`
    - Else: Return `(True, {"used": current, "limit": limit, "remaining": limit - current if limit > 0 else -1, "resets_at": self._next_pt_midnight_iso()})`
    - When limit == 0 (unlimited): remaining = -1 (sentinel for "unlimited")

    - Known services to always include: `[("gemini_image", "free"), ("gemini_text", "free")]`
