  <name>UsageAwareKeySelector</name>
  <files>src/services/key_selector.py, tests/test_key_selector.py</files>
  <behavior>
    ## KeyResolution dataclass
    - Frozen dataclass with fields: api_key (str), tier (str), mode (str)
    - tier values: "gemini_free" or "gemini_paid"
    - mode values: "auto", "forced_env", "forced_request", "free_only"

    ## Constructor behavior (D-01, D-02)
    - Reads GOOGLE_API_KEY, GOOGLE_API_KEY_PAID, GEMINI_FORCE_TIER from env
    - If GOOGLE_API_KEY_PAID is empty/missing: _free_only=True, logs warning "GOOGLE_API_KEY_PAID not configured -- free-only mode"
    - If GOOGLE_API_KEY_PAID == GOOGLE_API_KEY: _free_only=True, clears _paid_key, logs warning "GOOGLE_API_KEY_PAID identical to free key -- free-only mode"
    - If GOOGLE_API_KEY is empty: raises ValueError at construction time

    ## resolve() behavior (D-03, D-09)
    - Signature: async resolve(user_id: int, session: AsyncSession, force_tier: str | None = None) -> KeyResolution
    - Priority 1 (D-09): If force_tier is set, return forced resolution (mode="forced_request")
    - Priority 2 (D-09): If GEMINI_FORCE_TIER env is "free" or "paid", return forced resolution (mode="forced_env")
    - Priority 3: If _free_only, return free key (mode="free_only")
    - Priority 4: Call UsageRepository(session).check_limit(user_id, "gemini_image", "free")
      - If allowed=True: return free key (mode="auto", tier="gemini_free")
      - If allowed=False: return paid key (mode="auto", tier="gemini_paid"), log info with used/limit

    ## _forced_resolution() behavior
    - If tier="paid" but _free_only: log warning "Force paid requested but no paid key -- using free", return free key
    - If tier="paid" and paid key exists: return paid key
    - If tier="free": return free key

    ## Test cases (8 tests minimum)
    - test_resolve_returns_free_when_under_limit: mock check_limit returns (True, {...}) -> tier="gemini_free", mode="auto"
    - test_resolve_returns_paid_when_over_limit: mock check_limit returns (False, {...}) -> tier="gemini_paid", mode="auto"
    - test_free_only_mode_no_paid_key: no GOOGLE_API_KEY_PAID env -> always returns free, mode="free_only", never calls check_limit
    - test_free_only_mode_identical_keys: GOOGLE_API_KEY_PAID == GOOGLE_API_KEY -> free_only mode
    - test_force_tier_env_free: GEMINI_FORCE_TIER=free -> tier="gemini_free", mode="forced_env", no DB call
    - test_force_tier_env_paid: GEMINI_FORCE_TIER=paid -> tier="gemini_paid", mode="forced_env", no DB call
    - test_force_tier_request_overrides_env: force_tier param beats env var
    - test_force_paid_without_paid_key: force paid in free-only mode -> falls back to free with warning
    - test_resolution_tier_values: verify tier strings are exactly "gemini_free" or "gemini_paid"
    - test_empty_free_key_raises: no GOOGLE_API_KEY -> ValueError
  </behavior>
