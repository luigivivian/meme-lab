## Accomplishments
- Every generated phrase is now validated for length (<120 chars), language (Portuguese detection), and format (no placeholders) before use
- Rejected phrases trigger one automatic retry round to get valid replacements
- Topic-image coherence check prevents mismatched themes (e.g., "cafe" phrase with "confronto" background)
- Coherence mismatches trigger LLM-based theme remapping using lite tier for minimal cost
- All quality checks are non-blocking with graceful fallbacks
