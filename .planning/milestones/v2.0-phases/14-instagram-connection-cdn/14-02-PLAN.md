   - Calls `InstagramOAuthService(session).generate_auth_url()`
   - Returns `{"auth_url": str, "state": str}`
   - State is returned to frontend which stores it in sessionStorage for CSRF validation on callback

   **GET /instagram/callback** (per D-08):
   - Depends: get_current_user, db_session
   - Query params: code (str, required), state (str, optional — frontend validates CSRF client-side)
   - Calls `InstagramOAuthService(session).exchange_code(code=code, user_id=current_user.id)`
   - Returns `{"success": True, "ig_username": connection.ig_username, "connected_at": str(connection.connected_at)}`
   - On InstagramOAuthError: return 400 with error message
   - NOTE: This is called by the frontend after the OAuth popup redirects back. The frontend extracts code from URL params and calls this endpoint.

   **GET /instagram/status** (per D-09):
   - Depends: get_current_user, db_session
   - Calls `InstagramOAuthService(session).get_status(user_id=current_user.id)`
   - Returns the status dict or `{"connected": False}` if no connection

   **POST /instagram/disconnect** (per D-10):
   - Depends: get_current_user, db_session
   - Calls `InstagramOAuthService(session).disconnect(user_id=current_user.id)`
   - Returns `{"success": True}` or 404 if no connection found

   **POST /instagram/upload-media** (per D-11, D-01, D-02):
   - Depends: get_current_user, db_session
   - Body: `{"content_package_id": int}` (Pydantic model UploadMediaRequest)
   - Loads ContentPackage by id, verifies ownership (via pipeline_run.character.user_id or direct query)
   - Gets the image file path from content_package.image_path
   - Uploads to GCS via `GCSUploader(bucket_name="meme-lab-bucket")` — per D-01 use existing bucket
   - remote_name = `instagram-media/{content_package_id}/{filename}` (organized by package)
   - GCSUploader.upload_image() returns a signed URL (1-hour expiry per D-02)
   - Returns `{"cdn_url": signed_url, "content_package_id": id, "expires_in_seconds": GCS_SIGNED_URL_EXPIRY}`
   - On file not found: return 404
   - On GCS error: return 500

   Define Pydantic request model `UploadMediaRequest` with `content_package_id: int` at the top of the file.

   ```python
   async def _refresh_instagram_tokens():
       """Check and refresh Instagram tokens expiring within 7 days (per D-05)."""
       from src.database.session import get_session_factory
       from src.services.instagram_oauth import InstagramOAuthService

       try:
           session_factory = get_session_factory()
           async with session_factory() as session:
               service = InstagramOAuthService(session)
               refreshed = await service.refresh_expiring_tokens(days_before_expiry=7)
               if refreshed > 0:
                   logger.info(f"Instagram tokens refreshed: {refreshed}")

           logger.error(f"Instagram token refresh failed: {e}")
   ```
