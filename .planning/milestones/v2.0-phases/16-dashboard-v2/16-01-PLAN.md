**Step 1: Add two new methods to UsageRepository in usage_repo.py.**

Add `get_usage_history(self, user_id: int, days: int = 30) -> list[dict]`:
- Query `api_usage` WHERE user_id=user_id AND status='success' AND date >= (today_start - timedelta(days))
- GROUP BY date, service — SELECT date, service, SUM(usage_count) as count
- Build a list of dicts: `[{"date": "2026-03-01", "gemini_text": 45, "gemini_image": 3, "kie_video": 2}, ...]`
- Fill missing dates with zeros (generate all 30 dates, default services to 0)
- Use `_get_pt_today_start_utc()` for today reference and compute date range
- Sort by date ascending
- Format date as ISO date string (YYYY-MM-DD)

Add `get_cost_breakdown(self, user_id: int, days: int = 30) -> list[dict]`:
- Query `api_usage` WHERE user_id=user_id AND status='success' AND date >= (today_start - timedelta(days))
- GROUP BY service — SELECT service, SUM(cost_usd) as total_cost, SUM(usage_count) as total_calls
- Return `[{"service": "gemini_text", "cost_usd": 0.45, "calls": 120}, ...]`
- Only include services with cost > 0 or calls > 0

**Step 2: Create `src/api/routes/dashboard.py`** with `router = APIRouter(prefix="/dashboard", tags=["Dashboard"])`.

Per D-05: `GET /dashboard/usage-history?days=30`
- Depends: get_current_user, db_session
- Uses UsageRepository.get_usage_history(user.id, days)
- Returns `{"days": 30, "history": [{"date": "...", "gemini_text": N, ...}]}`

Per D-06: `GET /dashboard/cost-breakdown?days=30`
- Uses UsageRepository.get_cost_breakdown(user.id, days)
- Returns `{"days": 30, "services": [{"service": "...", "cost_usd": N, "calls": N}], "total_cost_usd": N}`

Per D-07: `GET /dashboard/pipeline-activity?days=30`
- Query PipelineRun directly with SQLAlchemy (not via repo — repo lacks date-range aggregate)
- SELECT func.date(PipelineRun.started_at) as day, func.count() as runs, func.sum(PipelineRun.packages_produced) as packages
- WHERE started_at >= (now - timedelta(days)) AND character linked to user (JOIN Character on character_id WHERE Character.user_id = user.id, or character_id IS NULL for legacy)
- GROUP BY day, ORDER BY day ASC
- Fill missing dates with zeros
- Returns `{"days": 30, "activity": [{"date": "...", "runs": N, "packages": N}]}`

Per D-08: `GET /dashboard/publishing-stats`
- Query ScheduledPost joined with Character for tenant scoping (same pattern as schedule_repo)
- SELECT status, func.count() — GROUP BY status
- Returns `{"total": N, "published": N, "queued": N, "failed": N, "cancelled": N}`
