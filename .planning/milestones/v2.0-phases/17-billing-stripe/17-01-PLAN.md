3. **Helper function** `get_tier_limit(plan_tier: str, service: str, tier: str) -> int` that looks up from PLAN_TIERS, falling back to the free plan limits. Returns 0 for unlimited.

4. **Class `StripeBillingService`** with `__init__(self, session: AsyncSession)`:

   a. `async create_checkout_session(user: User, price_id: str, success_url: str, cancel_url: str) -> str`:
      - If not is_stripe_configured(), raise ValueError("Stripe not configured")
      - Import stripe, set stripe.api_key = STRIPE_SECRET_KEY
      - If user has no stripe_customer_id, create a Stripe Customer (stripe.Customer.create with email=user.email, metadata={"user_id": str(user.id)}) and save the customer_id to user.stripe_customer_id, flush session
      - Create stripe.checkout.Session.create with mode="subscription", customer=user.stripe_customer_id, line_items=[{"price": price_id, "quantity": 1}], success_url, cancel_url
      - Return session.url

   b. `async create_portal_session(user: User, return_url: str) -> str`:
      - If not is_stripe_configured() or not user.stripe_customer_id, raise ValueError
      - Create stripe.billing_portal.Session.create with customer=user.stripe_customer_id, return_url
      - Return session.url

   c. `async handle_webhook_event(payload: bytes, sig_header: str) -> dict`:
      - Verify signature: stripe.Webhook.construct_event(payload, sig_header, STRIPE_WEBHOOK_SECRET)
      - Dispatch by event.type (per D-06):
        - `checkout.session.completed`: Extract customer ID and subscription ID from event.data.object. Find user by stripe_customer_id. Update user.stripe_subscription_id, plan_tier (map price_id to tier via STRIPE_PRO_PRICE_ID/STRIPE_ENTERPRISE_PRICE_ID), subscription_status="active". Flush.
        - `customer.subscription.updated`: Find user by subscription ID. Update subscription_status from event.data.object.status. If status is "active", ensure plan_tier matches price. If status is "past_due", keep current tier (grace period per BILL-05).
        - `customer.subscription.deleted`: Find user by subscription ID. Set plan_tier="free", subscription_status="canceled", subscription_ends_at=now. Flush.
        - `invoice.payment_failed`: Find user by customer ID. If subscription_status != "past_due", set subscription_status="past_due" (grace period). Flush.
      - Return {"status": "ok", "event_type": event.type}

   d. Private helper `async _find_user_by_stripe_customer(customer_id: str) -> User | None`: query User where stripe_customer_id == customer_id.

   e. Private helper `async _find_user_by_subscription(subscription_id: str) -> User | None`: query User where stripe_subscription_id == subscription_id.

   f. Private helper `_price_id_to_tier(price_id: str) -> str`: Maps STRIPE_PRO_PRICE_ID -> "pro", STRIPE_ENTERPRISE_PRICE_ID -> "enterprise", else "free".

5. **Graceful degradation (D-19):** All methods that call Stripe SDK check `is_stripe_configured()` first. The `get_tier_limit()` function works without Stripe — it just reads from PLAN_TIERS based on the user's plan_tier field (which defaults to "free").

Import stripe at function level (not module level) so the app works even without the stripe package installed when STRIPE_SECRET_KEY is empty.
