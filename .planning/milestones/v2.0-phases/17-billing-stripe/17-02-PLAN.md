1. Create `src/api/routes/billing.py` with `router = APIRouter(prefix="/billing", tags=["Billing"])` following the exact pattern from `dashboard.py`.

2. **GET /billing/status** (per D-14) — authenticated via `Depends(get_current_user)`:
   - Import `PLAN_TIERS, is_stripe_configured, get_tier_limit` from `src.services.stripe_billing`
   - Import `UsageRepository` from `src.database.repositories.usage_repo`
   - Read user's `plan_tier`, `subscription_status`, `subscription_ends_at`, `stripe_subscription_id`
   - Get today's usage via `UsageRepository(session).get_user_usage(current_user.id)`
   - For each service in the usage response, override the `limit` field with `get_tier_limit(user.plan_tier, service, tier)` — this makes the limit tier-aware
   - Return:
     ```json
     {
       "plan": "free",
       "plan_name": "Free",
       "subscription_status": null,
       "subscription_ends_at": null,
       "stripe_configured": true/false,
       "services": [...],  // with tier-aware limits
       "resets_at": "..."
     }
     ```

3. **POST /billing/create-checkout** (per D-12) — authenticated:
   - Accept JSON body: `{"price_id": str, "success_url": str, "cancel_url": str}`
   - If not `is_stripe_configured()`, return 503 with `{"detail": "Billing not configured. Set STRIPE_SECRET_KEY in .env"}`
   - Validate price_id is one of STRIPE_PRO_PRICE_ID or STRIPE_ENTERPRISE_PRICE_ID, else 400
   - Create StripeBillingService(session), call create_checkout_session
   - Return `{"checkout_url": url}`

4. **POST /billing/webhook** (per D-13) — NO auth (Stripe calls this directly):
   - Read raw body via `request.body()` (not JSON parsed)
   - Read `stripe-signature` header
   - If not `is_stripe_configured()`, return 503
   - Create StripeBillingService(session), call handle_webhook_event(payload, sig_header)
   - Return 200 with result dict
   - Wrap in try/except: on ValueError (bad signature) return 400, on Exception return 500

5. **POST /billing/portal** (per D-15) — authenticated:
   - Accept JSON body: `{"return_url": str}`
   - If not `is_stripe_configured()`, return 503
   - If user has no stripe_customer_id, return 400 with `{"detail": "No billing account. Subscribe to a plan first."}`
   - Create StripeBillingService(session), call create_portal_session
   - Return `{"portal_url": url}`

6. **Wire to app.py**: In `src/api/app.py`:
   - Add `billing` to the import line: `from src.api.routes import ..., billing`
   - Add `app.include_router(billing.router)` after the dashboard router line

IMPORTANT: The webhook endpoint MUST use `Request` from fastapi to read raw bytes, NOT a Pydantic model. Stripe signature verification requires the raw body. Use:
```python
from fastapi import Request
@router.post("/webhook")
async def stripe_webhook(request: Request, session: AsyncSession = Depends(db_session)):
    payload = await request.body()
    sig_header = request.headers.get("stripe-signature", "")
```
