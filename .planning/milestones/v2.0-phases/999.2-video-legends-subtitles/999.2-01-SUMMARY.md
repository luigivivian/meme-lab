## Accomplishments
- LegendRenderer class renders text overlays on video using FFmpeg drawtext filter with 3 modes matching image_maker.py visual style
- Word-wrap function ported from image_maker.py using identical Pillow font metrics (getbbox) ensures consistent text layout between image and video
- Windows path escaping (forward slashes, colon escape) and textfile approach for UTF-8 safety
- Audio stream detection with -an fallback handles Sora 2 videos without audio track
- Config constants with env var support, DB migration, and ORM columns for legend tracking
