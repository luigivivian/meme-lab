## Decisions Made
- Direct Gemini API calls (not wrapping GeminiImageClient) keeps image_gen simple and avoids wrapper-on-wrapper anti-pattern
- ROTEIRO_SCHEMA as dict (not Pydantic) passed to response_schema -- Gemini API expects dict format
- PCM 24kHz mono 16-bit from Gemini TTS wrapped with Python wave module (no ffmpeg dependency for audio)
- Markdown code fence stripping in transcriber handles cases where Gemini wraps SRT output
