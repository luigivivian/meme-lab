Create `src/reels_pipeline/video_builder.py` with:

**`def is_ffmpeg_available() -> bool`**:
- Check `shutil.which("ffmpeg")` is not None (same pattern as legend_renderer.py)

**`def build_reel_video(image_paths: list[str], audio_path: str, srt_path: str, output_path: str, config_override: dict | None = None) -> str`**:
- Check FFmpeg available, raise RuntimeError if not
- Read config values from config_override merged with REELS_* defaults (image_duration, transition_duration, transition_type, fps, crf)
- Build the FFmpeg command using xfade filter (per RESEARCH Pattern 3, NOT concat demuxer which lacks crossfade):

1. **Inputs**: For each image, add: `-loop 1 -t {image_duration} -framerate {fps} -i {img_path}`
   Then add audio: `-i {audio_path}`

2. **Scale filters**: For each image index i:
   `[{i}]scale=1080:1920:force_original_aspect_ratio=decrease,pad=1080:1920:-1:-1:color=black[s{i}]`

3. **xfade chain**: Chain xfade transitions between consecutive scaled outputs:
   - For i from 1 to len(images)-1:
   - offset = i * image_duration - i * transition_duration
   - prev label = "s0" for first, "f{i}" for subsequent
   - out label = "f{i}" for intermediate, "vout" for last
   - `[{prev}][s{i}]xfade=transition={transition_type}:duration={transition_duration}:offset={offset}[{out}]`
   - If only 1 image: skip xfade, use `[s0]` directly as `[vout]` via copy: `[s0]copy[vout]`

4. **Subtitle overlay**: `[vout]subtitles={srt_path}:force_style='FontSize=52,PrimaryColour=&HFFFFFF&,OutlineColour=&H000000&,Outline=2,Alignment=2,MarginV=200'[final]`
   - Escape path separators for FFmpeg on all platforms (replace backslash with forward slash, escape colon)

5. **Output args**: `-map [final] -map {audio_index}:a -c:v libx264 -preset fast -crf {crf} -c:a aac -b:a 192k -r {fps} -shortest -movflags +faststart {output_path}`

- Combine filter_complex as semicolon-joined string of scale_filters + xfade_filters + subtitle_filter
- Run via `subprocess.run(cmd, capture_output=True, text=True, timeout=300)`
- If returncode != 0, raise RuntimeError with stderr
- os.makedirs for output parent directory
- Return output_path

**`def get_video_duration(video_path: str) -> float`**:
- Run `ffprobe -v error -show_entries format=duration -of csv=p=0 {video_path}`
- Return float(stdout.strip())

Create `src/reels_pipeline/main.py` — the sequential orchestrator (per D-01).

**`@dataclass class ReelsResult`**:
- job_dir: str
- video_path: str
- script: dict (the roteiro JSON)
- image_paths: list[str]
- audio_path: str
- srt_path: str
- cost_usd: float
- cost_brl: float

**`class ReelsPipeline`**:
- `__init__(self, config_override: dict | None = None)` — merges DB config with REELS_* defaults
- `async def run(self, tema: str, character_id: int | None = None, on_progress: Callable | None = None) -> ReelsResult`:

The on_progress callback signature: `(step: str, pct: int) -> None` — used by API route to update job status.

Sequential steps:
1. **Setup**: Create job directory at `{REELS_OUTPUT_DIR}/{timestamp}_{tema_slug}/` with subdirs `images/`
2. **Step 1 - Images** (0-20%): Call `generate_reel_images(tema, character_id, job_dir/images, count, config)` -> image_paths. Call on_progress("images", 20).
3. **Step 2 - Script** (20-40%): Call `generate_script(image_paths, tema, config)` -> script dict. Save as job_dir/roteiro.json. Call on_progress("script", 40).
4. **Step 3 - TTS** (40-60%): Call `generate_narration(script["narracao_completa"], job_dir/audio.mp3, voice, speed, provider)` -> audio_path. Call on_progress("tts", 60).
5. **Step 4 - Transcription** (60-75%): Call `transcribe_to_srt(audio_path, job_dir/subtitles.srt, language, provider)` -> srt_path. Call on_progress("transcription", 75).
6. **Step 5 - Assembly** (75-95%): Call `build_reel_video(image_paths, audio_path, srt_path, job_dir/final.mp4, config)` -> video_path. Call on_progress("assembly", 95).
7. **Done** (100%): Call on_progress("complete", 100).

Cost tracking: Accumulate from estimate_tts_cost + estimate_transcription_cost. Image gen cost ~$0.00 (free tier). Script ~$0.001. BRL = USD * REELS_USD_TO_BRL.

Error handling: If any step fails, log the error and re-raise. The API route catches and updates job status to "failed".

Return ReelsResult with all paths and costs.
