## Decisions Made
- Prompt step runs synchronously (just sets up job directory), heavy steps (images, script, tts, srt, video) run as background tasks
- flag_modified(job, "step_state") on every step_state mutation for reliable JSON persistence
- Regenerate clears artifact fields and approval for all downstream steps before re-executing
- File serving uses os.path.realpath to prevent path traversal attacks
- SRT edit reconstructs the .srt file from structured entries
