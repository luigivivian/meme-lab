# Plan 01 — Loop phrase generation + video assembly

## Goal
Add a `frase_loop` field to the script schema so Gemini generates an explicit bridge phrase, and modify video assembly to append the first scene clip at the end with the loop phrase audio — creating a seamless infinite loop for Reels/TikTok/Shorts.

## Tasks

### Task 1: Add frase_loop to script generation
**Files:** `src/reels_pipeline/script_gen.py`
**Action:**
1. Add `frase_loop` (STRING) to `ROTEIRO_SCHEMA` as a required field
2. Update the LOOP VERBAL instruction in ALL language system prompts to emphasize generating `frase_loop` as a separate short phrase (3-8 words) that creates curiosity and connects back to the hook
3. In `generate_script()`, after getting the result: if `frase_loop` exists, append it to `narracao_completa` (if not already there)

**Verify:** Script schema includes `frase_loop`, system prompt references it, post-processing appends to narracao_completa.
**Done:** `frase_loop` flows through the pipeline — TTS includes it in audio, SRT covers it.

### Task 2: Video builder — append loop clip at end
**Files:** `src/reels_pipeline/video_builder.py`
**Action:**
1. In `concat_clips_with_audio`, after computing `scene_durs` (both script_json and SRT paths), check if `script_json` has a non-empty `frase_loop`
2. If yes:
   - Append `clip_paths[0]` (first scene clip) to `clip_paths` — this creates the visual loop by showing the first scene again
   - Add a synthetic cena entry `{"narracao": frase_loop, ...}` to a working copy of cenas for proportional duration computation
   - Recompute `scene_durs` with the extended clip count
3. The rest of the pipeline (trim, xfade, subtitles) handles it naturally since clip count matches scene count

**Verify:** When frase_loop exists, final video has N+1 clips (last = first scene repeated). When absent, behavior unchanged.
**Done:** Video ends with first scene visual + loop phrase audio, creating seamless loop.
