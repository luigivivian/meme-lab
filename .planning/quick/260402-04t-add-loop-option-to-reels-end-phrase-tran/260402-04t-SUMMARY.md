# Quick Task 260402-04t: Add loop option to reels

## What was done

Added a `frase_loop` field to the reels script generation schema and modified the video assembly pipeline to create seamless infinite loop videos for Reels/TikTok/Shorts.

### Task 1: Script generation (`src/reels_pipeline/script_gen.py`)
- Added `frase_loop` (STRING, required) to `ROTEIRO_SCHEMA`
- Updated LOOP VERBAL technique instructions in all 4 language templates (pt-BR, en-US, es-ES, fallback) to explicitly request a short 3-8 word bridge phrase
- Post-processing: `frase_loop` is automatically appended to `narracao_completa` so TTS includes it in the audio

### Task 2: Video assembly (`src/reels_pipeline/video_builder.py`)
- In `concat_clips_with_audio`, when `script_json` has `frase_loop`:
  - Copies the first scene clip as `clip_XX_loop.mp4` (avoids overwriting the first scene's trimmed file)
  - Appends it to clip_paths as the final scene
  - Adds a synthetic cena entry so proportional duration computation includes the loop segment (~3s)
- Result: final video ends with the first scene's visual + loop phrase audio, creating seamless loop when video replays
