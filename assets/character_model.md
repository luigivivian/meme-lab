## 7. Workflow Recomendado para Gerar Backgrounds

### Passo 1 — Gerar Imagem Ancora (1x)
Gere a **Pose Neutra (3.1)** em fundo branco. Esta sera sua referencia principal.

### Passo 2 — Gerar Variacoes de Cenario
Usando a imagem ancora como `--cref` (Midjourney) ou Character Reference (Leonardo AI),
gere cada cenario da tabela acima mantendo o personagem identico.

### Passo 3 — Composicao Final
O `image_maker.py` do projeto adiciona automaticamente:
- Overlay azul noturno (para uniformizar tons entre backgrounds diferentes)
- Vinheta escura nas bordas
- Glow dourado central
- Texto com contorno preto
- Watermark @omagomestre

Isso significa que backgrounds podem ter estilos levemente diferentes —
o pipeline de composicao vai padronizar o visual final.

### Passo 4 — Salvar em assets/backgrounds/
Nomeie os arquivos descritivamente:
```
mago_floresta_mistica_01.png
mago_torre_estudando_01.png
mago_taverna_relaxando_01.png
```

## 8. Checklist de Consistencia

Antes de usar qualquer imagem gerada, verifique:

- [ ] Barba branca longa ondulada (nao curta, nao lisa)
- [ ] Chapeu pontudo cinza-azulado com aba larga
- [ ] Tunica azul noturno escuro (nao branca, nao marrom)
- [ ] Detalhes dourados na tunica (sutil, nao exagerado)
- [ ] Cajado de madeira retorcida com brilho dourado no topo
- [ ] Olhos azuis claros visiveis
- [ ] Expressao comica/acolhedora (NUNCA ameacador ou assustador)
- [ ] Personagem no terco inferior da imagem
- [ ] Area superior livre para texto
- [ ] Proporcao 4:5 vertical (1080x1350)
- [ ] Estilo cartoon semi-realista (nao foto, nao anime)

## 9. O Que NAO Fazer

- **NAO** gerar o mago como realista/fotorrealista (deve ser cartoon semi-realista)
- **NAO** usar estilo anime/manga/chibi
- **NAO** fazer o mago parecer ameacador, assustador ou maligno
- **NAO** trocar as cores da tunica (SEMPRE azul noturno escuro)
- **NAO** remover o chapeu (elemento icônico de identidade)
- **NAO** encurtar a barba (SEMPRE longa ate o peito)
- **NAO** fazer o mago jovem (SEMPRE idoso, 70-80 anos)
- **NAO** colocar o personagem no centro/topo (SEMPRE terco inferior)
- **NAO** usar fundos muito claros/brancos (SEMPRE atmosfera escura/mistica)
